# 🐍 贪吃蛇 Snake Game

一个使用 Python 内置 `turtle` 模块实现的经典贪吃蛇游戏，无需安装任何第三方依赖。

## 运行方式

```bash
python main.py
```

## 操作说明

| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` | 控制蛇的移动方向 |
| `W` `A` `S` `D` | 方向键替代方案 |
| `Space` | 游戏结束后重新开始 |
| `Esc` | 退出游戏 |

## 游戏规则

- 🍎 吃到红色食物得 10 分
- 🧱 撞墙游戏结束
- 🐍 撞到自己游戏结束
- ⚡ 分数越高，蛇移动越快
- 🏆 最高分自动保存到 `high_score.txt`

## 技术栈

- Python 3
- `turtle`（内置模块）
- `random`（内置模块）
