# 🐍 贪吃蛇 Snake Game

一个使用原生 **HTML / CSS / JavaScript** 实现的网页贪吃蛇游戏，功能丰富，零依赖，打开浏览器即可游玩。

## 🚀 运行方式

**最简单**：双击 `index.html`，浏览器直接打开。

或者用 HTTP 服务器：

```bash
python -m http.server 8080     # Python 3
npx serve .                    # Node.js
```

然后访问 `http://localhost:8080`。

---

## 🎮 操作说明

| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` | 控制蛇的移动方向 |
| `W` `A` `S` `D` | 方向键替代方案 |
| `Space` | 游戏结束后重新开始 |
| `P` | 暂停 / 继续 |
| `M` | 开关音乐 / 音效 |
| 滑块 0.5x – 2.5x | 调节游戏速度 |
| ↻ 按钮 | 重置最高分 |

## 🍎 食物系统

游戏采用**双轨食物机制**——普通苹果始终在场，特殊道具独立出现，两者可共存。

| 食物 | 外观 | 分值 | 规则 |
|------|------|------|------|
| 🍎 Apple | 红色圆形 | +10 | 始终存在，吃掉立即刷新 |
| ⭐ Golden | 金色五角星 | +30 | 随机出现，7 秒后消失 |
| 🍒 Cherry | 粉色爱心 | +50 | 随机出现，5 秒后消失 |
| 💀 Poison | 紫色菱形 | −5 | 得分 ≥ 30 后才出现，6 秒后消失 |

- 特殊道具每隔 8–15 秒随机生成一个，和红苹果同时在场
- 特殊道具位置不和蛇身 / 红苹果重叠
- 倒计时归零自动消失，最后 2 秒闪烁提示
- 外围倒计时环直观显示剩余时间

---

## 🔥 Combo 连击系统

2 秒内连续吃食物即触发连击：

- 每次连击 +1 层，额外加分 = `(combo − 1) × 5`
- 例：连吃 3 个 → +10, +15, +20
- Combo ≥ 3：侧边栏数字闪烁
- Combo ≥ 5：画布边框金色脉冲发光

---

## ⚡ 速度系统

- **基础加速**：每吃一个食物，蛇的移动间隔减少 2ms（120ms → 最快 40ms）
- **手动调速**：左侧滑块 0.5x – 2.5x，步进 0.1x
- 两个速度叠加：`实际速度 = 基础间隔 ÷ 手调倍率`

---

## 🏆 最高分

- 通过浏览器 `localStorage` 持久化存储
- 刷新页面 / 关闭浏览器 / 重启电脑均不丢失
- 别人的电脑打开会从 0 开始（数据存在本地）
- 点分数栏右侧 ↻ 按钮可重置

### 🎉 新高分庆祝

当一整局游戏结束时，如果本局分数**超过了历史最高分**，GAME OVER 界面会被庆祝画面取代：

- 🏆 金杯图标 + 缩放弹入动画
- ✨ "NEW HIGH SCORE!" 金色发光文字
- 🎉 100 片彩色礼花从天而降（重力 + 旋转）
- 🎵 上行琶音庆祝音效
- 不影响游戏过程（赛中不弹窗，结算时才判定）

---

## 🎵 音频

- 🎶 8-bit 风格循环背景音乐（Web Audio API 方波 + 三角波合成）
- 💎 吃食物：上行琶音（分值越高音符越多）
- 💀 游戏结束：下行悲鸣
- 🏆 新高分：大和弦上行琶音
- 🔊 右上角按钮或 `M` 键随时开关
- 快速重开时上一局残留音效自动切断，不会重叠

---

## ✨ 视觉特效

- 蛇身从亮青 `#00d2ff` 到深蓝 `#023e8a` 平滑渐变（不循环，多长都好看）
- 吃食物爆出彩色粒子（数量随食物分值变化）
- 浮动分数文字飘升淡出（含 combo 倍率信息）
- 特殊食物各有专属形状（星形 / 爱心 / 菱形）
- 暂停 / 结束均有半透明遮罩覆盖

---

## 🛠 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 Canvas | 游戏渲染 |
| CSS3 | 界面布局与样式 |
| Web Audio API | 8-bit 音乐与音效合成 |
| localStorage | 最高分持久化 |
| requestAnimationFrame | 固定时间步长游戏循环 |

**无任何第三方框架、库或依赖。**
